# Picselit

Picselit is a web-based pixel art planner built for perler/fuse bead projects. Upload any image, and it will be converted into a pixel grid to help you track and complete your physical bead projects.

## Features
- **Upload & Pixelate**: Convert photos into color-reduced pixel art automatically.
- **Color Adjustments**: Change brightness, contrast, saturation, and vibrancy in real time.
- **Color Palette**: View all unique colors used in the generated image and highlight them on the canvas.
- **Interactive Canvas**: Use the canvas to mark your progress as you place your beads!
  - **Left Click**: Mark individual beads
  - **Left Click + Drag**: Mark a rectangle of beads at once
  - **Space + Drag** (or Alt+Drag / Middle Click): Pan/reposition the canvas
  - **Scroll**: Zoom in and out
- **Exporting**: Save your work as a normal PNG, a PNG showing your progress marks, or a scaled-up printable version.

## Running Locally

Run the development server:
```bash
npm run dev
```
